Camp Import

Files that are imported must have a .csv extension as such make sure that all your values are separated by commas.
It is important to note that when you specify an camp name that already exists in BSA Database this will update the existing data. To keep to existing scout participation list you must specify their names in the import.
When adding scout names please list all the scouts that have participated in a camp below that camp they participated in. To signify a new camp leave an empty space between the last camp or scout name and the name of the next camp.
The following table will explain what data is valid and how to build our import file. The type and max values are relate to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Camp Name (Required)
(type, max)		 The name of the camp.
(Text, 90)
Camp Location (Required)
(type, max)		 The actual location of the camp (address, etc.).
(Text, 254)
Camp Duration (Required)
(type, max)		 The whole number count of the number of nights spent on the camp.
(Numeric, N/A)
Camp Date (Required)
(type, max)		 The Date of the first night of the camp out. valid format is: MM/DD/YYYY
(Date, N/A)
Camp Notes (Optional)
(type, max)		 Any notes you might want to add to the camp out.
(Text, N/A)
Scout Name (Optional)
(type, max)		 The Name of the scout that participated in the camp out. The scout must exists in BSA Database
(Text, 90)

"Camp Name","Camp Location","Camp Duration","Camp Date","Camp Notes"
