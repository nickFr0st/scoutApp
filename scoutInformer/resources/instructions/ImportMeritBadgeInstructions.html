Merit Badge Import

Files that are imported must have a .csv extension as such make sure that all your values are separated by commas.
It is important to note that when you specify a merit badge name that already exists in BSA Database this will update the existing data. To keep existing requirements or contacts you must specify them in the import.
When adding counselors please list all counselors for a merit badge after the merit badge, also when adding a counselor put an asterisks(*) at the start of the name to signify that it is a counselor not a requirement.
When adding requirements please list all requirements for an merit badge after the merit badge. To signify a new merit badge leave an empty space between the last requirement and the name of the next merit badge.
The following table will explain what data is valid and how to build our import file. The type and max values are relate to the type of information that will be stored in the associated field (such as: Text or Number) and the maximum number of character allow in each field respectfully.

Merit Badge Name (Required)
(type, max)		 The name of the name of merit badge.
(Text, 254)
Is Required For Eagle (Required)
(type, max)		 If the merit badge is required for eagle then type true, else type false.
(Boolean, N/A)
Revision Date (Required)
(type, max)		 This is the revision date of your version of the merit badge. valid format: MM/DD/YYYY
(Date, N/A)
Merit Badge Image Path (Optional)
(type, max)		 The url of the merit badge image. The path should start at the c:/ directory.
(Text, 254)
Counselor Name (Optional)
(type, max)		 The name of the merit badge counselor.
(Text, 90)
Counselor Phone Number (Do not add if counselor name was not given)
(type, max)		 The merit badge counselors phone number.
(Text, 20)
Requirement Name (Optional)
(type, max)		 The name of the requirement.
(Text, 44)
Requirement Description (Do not add if requirement name was not given)
(type, max)		 The requirement description.
(Text, N/A)

Example

"Merit Badge Name", "Is Required For Eagle", "Revision Date", "Merit Badge Image Path"
"Counselor Name", "Counselor Phone Number"
"Requirement Name", "Requirement Description"
